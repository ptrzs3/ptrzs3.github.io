YAML Ain’t Markup Language Link to heading

  • 缩进几个空格字符不重要，同一层次的元素左侧要对齐

  • 缩进不能使用Tab字符，一般缩进两个空格

  • 可以增加空行以提升可读性

  • 同一个文件中，可以用---分隔

  • ...可用来表示文件结尾

  • 数组：可用区块格式（短杠+空格表示起始），或者使用内置格式

    --- # 最喜爱的电影
movies:
  - 肖申克的救赎
  - 阿甘正传
  - 霸王别姬
--- # 等价于
movies: [肖申克的救赎, 阿甘正传, 霸王别姬]

  • 对象：键值对集合

    --- # 区块格式
person:
  name: John Smith
  age: 33
--- # 内置格式
person: {name: John Smith, age: 33}

  • 字符串：不需要包含在引号内

    # 多行字符串，一种为保留换行（使用|字符）
data: |
  There once was a man from Darjeeling
  Who got on a bus bound for Ealing
      It said on the door
      "Please don't spit on the floor"
  So he carefully spat on the ceiling  

# 或者使用折叠换行（使用>字符）
data: >
  Wrapped text
  will be folded
  into a single
  paragraph

  Blank lines denote
  paragraph breaks

  The newline character is \ # '\'表示下一行与本行是同一行
  converted to a whitspace character

  • 锚点和引用

    # '<<'表示合并到当前数据，'&'建立锚点，'*'用来引用锚点
animal: &id001
  run: True
  eat: meat
extends:
  <<: *id001

# 等价于
animal:
  run: True
  eat: meat
extends:
  run: True
  eat: meat

  • 纯量

    boolean:
  - True # true, TRUE
  - False # false, FALSE
float:
  - 3.1415926
  - 0.31415926e+10
int:
  - 12345
  - 0b1011_0111 #二进制
null:
  - nodeName: 'node'
  - parent: ~ # 使用'#'表示null
string:
  - hello
  - hello
    world # 等价于'hello world'
date:
  - 2024-03-05 # 日期必须符合ISO8601 yyyy-MM-dd
datetime:
  - 2024-03-05T16:09:15+08:00 # ISO8601格式
