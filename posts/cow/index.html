ToOwned Link to heading 

pub trait ToOwned {
    // 不是为类型起别名，而是要求Owned类型必须实现Borrow trait
    type Owned: Borrow<Self>;

    // Required method
    fn to_owned(&self) -> Self::Owned;

    // Provided method
    fn clone_into(&self, target: &mut Self::Owned) { ... }
}

// 示例代码
// https://rustwiki.org/zh-CN/src/alloc/str.rs.html#208
#[cfg(not(no_global_oom_handling))]
#[stable(feature = "rust1", since = "1.0.0")]
impl ToOwned for str {
    type Owned = String;
    #[inline]
    fn to_owned(&self) -> String {
        unsafe { String::from_utf8_unchecked(self.as_bytes().to_owned()) }
    }

    fn clone_into(&self, target: &mut String) {
        let mut b = mem::take(target).into_bytes();
        self.as_bytes().clone_into(&mut b);
        *target = unsafe { String::from_utf8_unchecked(b) }
    }
}

Cow（Clone on write） Link to heading

介绍 Link to heading

Cow可以提供对象的不可变引用，并提供方法在需要可变时进行克隆，并避免重复克隆；

  • 实现了Deref trait，因此对于Cow<T>可以直接调用T的不可变方法

  • .to_mut()方法可以获得一个具有所有权的可变借用，可以多次调用这个方法，但不一定会产生克隆，最多只产生一次克隆

    pub fn to_mut(&mut self) -> &mut <B as ToOwned>::Owned

  • .into_owned()方法创建新的具有所有权的对象，这意味着内存拷贝，并且调用此方法之后原来的对象就被move了

    // 参数是self
pub fn into_owned(self) -> <B as ToOwned>::Owned

类型签名 Link to heading 

// trait bound
// 要求B类型实现ToOwned trait，可以是DST类型，且肯定实现了Borrow trait
// 两个成员分别包裹一个B的引用类型值，和B作为ToOwned trait的Owned类型
pub enum Cow<'a, B>
where
    B: 'a + ToOwned + ?Sized,
{
    Borrowed(&'a B),
    Owned(<B as ToOwned>::Owned),
}
